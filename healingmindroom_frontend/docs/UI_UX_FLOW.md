# UI/UX Flow Specification - Healing Mind Room MVP

## 1. UI/UX 디자인 원칙
- **Easy & Clean (단순성)**: 30~60대 중장년층을 타깃으로 하는 만큼, 화면 하나당 하나의 주요 액션만 처리하도록 단계를 분할합니다. (4단계 싱글 페이지 플로우 구성)
- **High Readability (높은 가독성)**:
  - 기본 폰트 크기를 크게 지정합니다 (본문 16px, 제목 24px 이상).
  - 버튼 크기를 최소 48px 높이로 구성하여 터치 미스를 방지합니다.
- **Natural Concept (내추럴 테마)**:
  - 깊은 숲과 식물을 연상시키는 **허벌 그린(#2d6a4f)**, **내추럴 샌드 계열**의 컬러 팔레트를 사용합니다.
  - 글래스모피즘(Glassmorphism) 효과를 통해 정제되고 고급스러운 힐링 테마를 연출합니다.

## 2. 화면별 세부 인터랙션 설계

### 2.1 Step 1: 세대별 AI 상담 (Home)
- **비주얼**: 편안한 숲속 안개가 움직이는 듯한 은은한 CSS 그라데이션 배경.
- **구성 요소**:
  - 연령대 선택 칩 (30대 / 40대 / 50대 / 60대) - 큼직한 가로 배치 카드형 버튼.
  - 고민 입력 창 - 플레이스홀더를 통해 예시("요즘 직장 일로 몸과 마음이 너무 쉽게 지치고 잠을 잘 이루지 못합니다.")를 제공하여 입력을 유도.
  - 다음 단계 버튼 - 입력 완료 시 활성화되는 아늑한 녹색 버튼.
- **인터랙션**: 연령대를 선택하면 고민 입력 창이 부드럽게 나타나고(Fade-in), 50자 이상 고민 작성 시 "다음 단계로" 버튼이 활성화됨.

### 2.2 Step 2: 감정 자가진단 위젯 (Diagnostic)
- **비주얼**: 부드러운 파스텔 옐로우와 아늑한 조명 효과.
- **구성 요소**:
  - **피로도 슬라이더**: 1(가뿐함)부터 5(극심한 피로)까지 마우스/터치 슬라이더. 숫자 밑에 간단한 이모지(😊 → 😩) 표시.
  - **수면 상태 단일 선택**: 3가지 옵션(잘잠 ☀️ / 보통 ⛅ / 못잠 🌧)을 카드 형태로 구성.
  - **긴장도 단일 선택**: 3가지 옵션(낮음 🌱 / 보통 🌿 / 높음 🔥)을 카드 형태로 구성.
- **인터랙션**: 슬라이더를 밀거나 카드를 클릭할 때마다 상위 상태가 갱신되며, 모든 선택이 끝나면 "나만의 힐링처 분석하기" 버튼이 로딩 스피너와 함께 등장.

### 2.3 Step 3 & 4: 분석 편지 & 농장 매칭 결과 & 공간 원화 시각화 (Result)
- **비주얼**: 신비로운 퍼플 및 에메랄드 그린 컬러 테마.
- **구성 요소**:
  - **AI 감정 진단 편지**: 사용자의 고민 텍스트에서 분석된 감정 키워드(예: #번아웃, #가족, #수면장애)와 맞춤 위로 문구 출력.
  - **뇌파 추정 대시보드**: 알파파와 세타파의 추정 비율을 시각적인 원형 게이지(Gauge)나 깔끔한 바 그래프로 전시.
  - **맞춤 치유농장 리스트**: 스코어링이 가장 높은 농장 2~3곳 추천. 각 농장 정보 카드가 좌우 슬라이드(Carousel) 또는 세로 배치로 구성됨.
  - **공간 원화 시각화**: 추천된 농장 카드 내부에 플로우 AI가 그린 몽환적이고 평화로운 원화 아트워크가 넓게 채워짐. 마우스를 올리거나 터치하면 농장의 상세 힐링 프로그램 및 연락처가 부드럽게 오버레이(Hover Overlay)됨.
- **인터랙션**:
  - 결과 로딩 시 1.5초간 잔잔한 심호흡 애니메이션("숨을 들이쉬고 내쉬세요...")을 제공하여 대기 시간을 힐링 콘텐츠로 활용.
  - "카카오톡으로 힐링 공간 공유하기" 버튼과 "처음부터 다시 진단하기" 버튼 제공.

## 3. 화면 전환 흐름도
```
[Step 0: 소셜 로그인 (Login)]
      │ (카카오/구글 로그인 성공)
      ▼
[Home (Step 1)]
      │ (연령대 선택 & 고민 작성 완료)
      ▼
[Diagnostic (Step 2)]
      │ (슬라이더 및 상태 입력 완료 -> "분석하기" 클릭)
      ▼
[Loading Page] -> 심호흡 가이드 애니메이션 (1.5초)
      │
      ▼
[Result (Step 3 & 4)]
  - AI 상담 편지
  - 뇌파 분석 게이지
  - 치유 농장 2~3개 카드 목록 & 힐링 공간 원화 시각화
  - 공유 / 다시하기 / 로그아웃 액션
```
