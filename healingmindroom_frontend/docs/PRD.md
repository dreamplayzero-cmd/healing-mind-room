# Product Requirement Document (PRD) - Healing Mind Room MVP

## 1. 프로젝트 개요 & 목적 (Goal)
- **개요**: 바쁜 현대인, 특히 스트레스와 심리적 불안을 겪는 **30대~60대 중장년층**을 대상으로 하는 AI 기반 힐링 및 맞춤형 치유농장 매칭 플랫폼.
- **목적**: 
  - 사용자의 고민 글을 NLP(자연어 처리)로 분석하여 감정 상태를 진단합니다.
  - 슬라이더와 선택 버튼을 통한 간단한 신체/정신적 자가진단(피로도, 수면, 긴장도)으로 안정 상태(알파파) 및 피로 상태(세타파)를 추정합니다.
  - 분석 결과와 연령대 정보를 바탕으로 전국 120개 치유농장 DB 중에서 가장 적합한 2~3개의 치유농장을 매칭합니다.
  - Flow AI로 생성한 감성적인 공간 원화를 시각화하여 지친 마음에 즉각적인 시각적 위로를 제공합니다.

## 2. 타깃 사용자 (Target User)
- **핵심 타깃**: 30대, 40대, 50대, 60대의 직장인, 주부, 은퇴 준비자 등 스트레스 해소와 정서적 휴식이 필요한 중장년층.
- **사용자 특성**: 
  - 복잡한 모바일 UI나 지나치게 화려한 인터랙션보다는 가독성이 높고 사용법이 단순한 UI를 선호합니다.
  - 명확하고 큼직한 폰트, 이해하기 쉬운 뇌파 추정 슬라이더 조작 방식을 원합니다.

## 3. MVP 기능 범위 (Scope)

### 3.1 핵심 기능 (In-Scope)
- **Step 0: 소셜 로그인 (인증 가드)**
  - 카카오(Kakao) 및 구글(Google) 소셜 로그인 지원.
  - 로그인 세션 암호화 및 로컬스토리지 보관을 통한 로그인 상태 유지.
  - 프로필 이미지, 닉네임 정보를 헤더 영역에 바인딩하여 친근하고 개인화된 힐링 무드 조성.
- **Step 1: 세대별 AI 상담 (고민 분석)**
  - 30대/40대/50대/60대 중 연령대 선택.
  - 고민 사항 자유 텍스트 입력 (최대 500자).
  - NLP API 연동을 통한 핵심 감정 및 키워드 추출 (API 장애/미설정 시 로컬 키워드 매칭 백업 작동).
  - 연령대에 맞는 맞춤형 AI 위로 메시지 출력.
- **Step 2: 감정 자가진단 위젯 (뇌파 추정)**
  - 3차 패션 프로젝트의 위젯 입력 로직을 계승하여 감정 상태를 도출하는 자가입력 장치 제공.
  - 피로도(1~5단계 슬라이더), 수면 상태(잘잠/보통/못잠), 긴장도(높음/보통/낮음) 입력.
  - 입력값 기반 알파파(안정) 및 세타파(피로) 추정값 공식 계산.
- **Step 3: 치유농장 매칭**
  - 계산된 뇌파 추정값(알파파/세타파) + 선택한 연령대 정보를 매칭 알고리즘에 전달.
  - 120개 농장 정적 DB(`farms.json`)를 탐색하여 최적의 농장 2~3개 필터링 및 추천.
- **Step 4: 공간 원화 시각화**
  - 추천된 농장의 테마(안정/활동/자연)와 연결된 플로우AI 제작 원화 이미지 전시.
  - 각 농장의 핵심 치유 프로그램, 위치, 연락처 정보 카드 제공.

### 3.2 제외 및 보류 기능 (Out-of-Scope)
- **7세대 확장 보류**: 10~20대 및 70대 이상 세대는 MVP 단계에서 분석 대상에서 제외.
- **마이페이지 상세 대시보드 보류**: 로그인 상태만 제공하고, 추천 이력 보관 및 누적 마이페이지 대시보드는 향후 고도화 단계로 이관.
- **실시간 예약 및 결제**: 농장으로 직접 연락할 수 있는 전화번호 링크 및 외부 예약 사이트(홈페이지 등) 연결로 대체.
- **실제 EEG 장비 연동**: 실제 뇌파 측정은 진행하지 않고, 프론트엔드 자가진단 위젯의 추정 계산 로직으로 대체.

## 4. 사용자 흐름 (User Journey)
1. **[진입 페이지]**: 따뜻하고 평온한 웰컴 메시지, 서비스 소개 및 연령대 선택, 고민 입력 폼.
2. **[자가진단 페이지]**: 3가지 스트레스 지표(피로도, 수면, 긴장도)를 입력하는 직관적인 위젯.
3. **[결과 페이지]**: AI 상담사의 분석 편지, 뇌파 추정 그래프(알파/세타파), 매칭된 치유농장 2~3곳 추천 정보, 공간 원화 이미지 시각화.

## 5. 성공 기준 (Done-When)
- 백엔드 서버 없이 프론트엔드 자체 로직 및 로컬 DB를 사용하여 4단계 플로우가 100% 매끄럽게 작동해야 함.
- 30~60대 사용자가 조작하기 쉬운 반응형 웹 UI 구성.
- NLP API 예외 처리가 완벽하여 네트워크 오류 상태에서도 Mock 데이터로 정상 추천까지 도달해야 함.
