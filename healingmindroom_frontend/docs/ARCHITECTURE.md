# System Architecture - Healing Mind Room MVP Frontend

## 1. 아키텍처 개요
본 프로젝트는 백엔드 서버가 최소화되었거나 없는 상태에서 프론트엔드 자체적으로 핵심 기능을 수행하는 **"클라이언트 중심 하이브리드 아키텍처"**로 설계되었습니다.
이를 통해 복잡한 RAG 또는 Vector DB 인프라 구축 없이도 신속하게 작동하는 16일 개발 주기의 MVP를 실현합니다.

```
[클라이언트 브라우저 (Vite + React + TS)]
  │
  ├── [인증 및 공통 상태 (React Context & Hooks)]
  │     ├── UserState (로그인 사용자 프로필 정보, 세션 토큰)
  │     ├── UserStateGuard (인증 검사: user가 없으면 Step0 로그인 화면 강제)
  │     └── DiagnosticState (자가진단 점수, 계산된 뇌파값)
  │
  ├── [비즈니스 로직 모듈 (Pure TS Utils)]
  │     ├── emotionAnalyzer.ts (NLP API / 로컬 키워드 매칭)
  │     ├── waveEstimator.ts (자가진단 데이터 -> 뇌파 추정 알고리즘)
  │     └── matcher.ts (농장 DB 필터링 및 매칭 알고리즘)
  │
  └── [로컬 데이터베이스 (정적 JSON)]
        └── farms.json (전국 120개 치유농장 데이터)
```

## 2. 3차 프로젝트 위젯 로직의 계승 (Inheritance)
3차 패션 프로젝트에서 사용된 위젯 로직들을 본 힐링 플랫폼의 성격에 맞춰 다음과 같이 성공적으로 변환 및 계승합니다.

| 3차 패션 프로젝트 요소 | 4차 힐링 프로젝트 계승 요소 | 아키텍처적 변환 방식 |
|---|---|---|
| **Weather 위젯** (외부환경 분석) | **NLP 텍스트 분석** (내면환경 분석) | 외부 API 연동을 통해 수치(날씨 데이터) 또는 텍스트(감정 키워드)를 입력받아 분석하는 공통 로직 모듈 계승. |
| **Scanner 위젯** (소재 분석 / TPO 필터링) | **자가진단 위젯** (스트레스 분석) | 사용자의 직접적인 조작을 수치화하여 상태값(피로도, 긴장도)을 조합 및 가중치 합산하는 입력 위젯 구조 계승. |
| **MBTI 추천** (개인 선호도 매칭) | **연령대 및 뇌파 매칭** (개인 맞춤 추천) | 다차원 가중치 필터링 알고리즘을 사용해 최종 결과물(옷 스타일 vs 치유농장)을 스코어링하여 상위 N개 추천하는 구조 계승. |
| **색채심리학** (시각 표현) | **공간 원화 시각화** (감성 공간 표현) | 분석된 감정 테마에 맞춰 매핑된 고유한 이미지(플로우 AI 원화)와 테마별 컬러셋(CSS Variables)을 매핑하여 화면 테마를 전환하는 방식 계승. |

## 4. 핵심 모듈 상세 설계

### 4.1 뇌파 추정기 (`waveEstimator.ts`)
- **수식 알고리즘**:
  - 피로도 $F$ (1~5), 수면 상태 $S$ (잘잠=1, 보통=0.5, 못잠=0), 긴장도 $T$ (낮음=0, 보통=0.5, 높음=1)
  - 세타파(피로 뇌파) 추정치 = $(F \times 0.4) + ((1 - S) \times 0.3) + (T \times 0.3)$
  - 알파파(안정 뇌파) 추정치 = $1.0 - \text{세타파 추정치}$
  - 결과에 따라 사용자 상태를 `'피로형(Theta-dominant)'` 또는 `'안정형(Alpha-dominant)'`으로 최종 분류.

### 4.2 농장 매칭기 (`matcher.ts`)
- `farms.json` 내의 120개 농장을 순회하며 다음 기준의 적합도 점수(Score)를 계산합니다.
  - **연령대 매칭**: 농장의 `targetAges` 배열에 사용자의 연령대가 포함되면 +50점.
  - **뇌파 테마 매칭**: 뇌파 상태가 `Theta-dominant`일 때 농장 테마가 `안정` 또는 `자연`이면 +30점. `Alpha-dominant`일 때 농장 테마가 `활동` 또는 `자연`이면 +30점.
  - **감정 키워드 매칭**: 고민 키워드 분석 결과와 농장 태그 일치 시 개당 +10점.
  - 가장 점수가 높은 상위 3개 농장을 최종 추천 대상으로 선정.
